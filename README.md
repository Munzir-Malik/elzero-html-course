## Overview

This repository contains my work for the Elzero Web School HTML course.
The course lasts three weeks, and each week includes a set of assignments.
The repository is organized into folders for video assignments and website tasks.

It is recommended to attempt each assignment independently before reviewing the solutions in this repository.

### How to Use This Repository

* Clone the repository:

      git clone https://github.com/Munzir-Malik/elzero-html-course.git


* Open the folder for the week you want to explore.

* Review the assignment files and code structure.
