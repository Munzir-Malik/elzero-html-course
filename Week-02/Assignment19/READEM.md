# Semantic HTML Page

### Task

Create a complete web page using HTML Semantic Elements only.

### Requirements

* Create a full HTML page from scratch.

* Use semantic elements correctly in all parts of the page.

* Create a header that contains:

    The page title

    Five links with any names you choose

* Below the header, create a nav element that contains:

    At least six navigation links

* Create a main section that contains:

    A content area

  A sidebar next to it

* Inside the content area:

    Create three articles

    Separate each article with a horizontal line

* Each article must include:

    An article title

    A paragraph with part of the article content

    An image

    A link with the text “Read More”

* Create a footer at the bottom of the page that contains:

    The text: Copyright 2021 ©

    Use the correct HTML entity for the copyright symbol

* Inside the sidebar:

    Add a heading titled Categories

    Below it, add an unordered list with five category names
