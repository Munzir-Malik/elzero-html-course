# Video Element

### Task Description

Add a video element with multiple formats and advanced settings.

### Requirements

* Add a video element that contains at least three video extensions.

* Display a message if the browser does not support the video file.

* The video must:

    Play automatically when the page loads

    Be muted while playing

* Add captions in two different languages that the user can choose from.

* Add a poster image that appears while the video is loading.
